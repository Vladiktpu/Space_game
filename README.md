Название: Space Defender

Описание: Космическая аркада. Управляйте ракеткой мышью, отбивайте мяч и набирайте очки. 
Попадите мячом в мерцающую звезду на карте, чтобы получить дополнительные +50 очков.

Управление:
- Мышь: движение ракетки
- Кнопка "СТАРТ" на экране: начало игры
- N или R: новая игра
- ESC: выход

Цель: Набрать максимальное количество очков, не потеряв 3 жизни.
